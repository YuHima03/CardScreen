# CardScreen

## 概要

名刺を画面上の好きな位置に配置、表示するWebアプリです。

胸ポケットに入れたデバイスの画面上に表示したり、机の上に置いたデバイスにフルスクリーンで表示させるといった利用法を想定しています。

## 機能

- 名刺に表示できる要素
  - 氏名
  - 所属
  - アイコン
  - タグ (複数可)
- 名刺の表示
  - 表示位置、サイズの変更
